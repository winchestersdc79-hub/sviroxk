# 📱 Productivity App

Приложение для продуктивности на основе Матрицы Эйзенхауэра.

## Функции
- ✅ Матрица Эйзенхауэра (4 квадранта)
- ⏰ Дедлайны и напоминания
- 🍅 Pomodoro таймер
- 📊 Статистика продуктивности
- 🗄️ Архив выполненных задач
- 📎 Прикрепление файлов и фото

## Технологии
- Flutter
- Dart
