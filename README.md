# Sviroxk — Приложение для продуктивности

Мобильное приложение для студентов на базе Expo React Native. Улучшенная версия оригинального Flutter-приложения.

## Возможности

- **Матрица Эйзенхауэра** — управление задачами по 4 квадрантам (Сделать / Запланировать / Делегировать / Убрать)
- **Управление задачами** — приоритеты (П1/П2/П3), подзадачи, теги, дедлайны
- **Календарь** — месячный вид с визуализацией дедлайнов
- **Pomodoro-таймер** — режимы фокуса, короткого и длинного перерыва
- **Трекер привычек** — ежедневное отслеживание, серии и 7-дневная визуализация
- **Статистика** — процент выполнения, еженедельная активность, прогресс по квадрантам
- **Поиск** — фильтрация задач по тексту, приоритету, квадранту
- **Архив** — выполненные задачи с возможностью восстановления

## Технологии

- **Expo** (React Native)
- **expo-router** — файловая маршрутизация
- **AsyncStorage** — локальное хранение данных
- **React Context + useReducer** — управление состоянием
- **TypeScript**

## Запуск

```bash
npm install
npx expo start
```

Отсканируй QR-код в приложении **Expo Go** на телефоне.

## Дизайн

- Тёмная тема (#0D0D1F фон, фиолетовый акцент)
- Полностью на русском языке
- Поддержка iOS и Android
