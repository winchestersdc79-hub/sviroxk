# 📱 Sviroxk — Приложение для продуктивности

Приложение для продуктивности на основе **Матрицы Эйзенхауэра**, разработанное с использованием Flutter.

## ✨ Основные функции

- ✅ **Матрица Эйзенхауэра**: Разделение задач на 4 квадранта (Срочно/Важно).
- 📅 **Календарь**: Визуализация дедлайнов и планирование.
- 🔍 **Умный поиск**: Фильтрация по приоритетам, квадрантам и дедлайнам.
- 🍅 **Pomodoro таймер**: Техника концентрации с уведомлениями.
- ✅ **Трекер привычек**: Формирование полезных привычек со стриками.
- 🤖 **AI-помощник**: Советы по продуктивности и управлению временем.
- 📊 **Детальная статистика**: Анализ выполнения задач с графиками.
- 🗄️ **Архив**: Автоматическое хранение выполненных задач.
- ⚙️ **Настройки**: Тёмная тема, выбор цвета акцента, пин-код и язык.

## 🛠 Технологии

- **Flutter / Dart**
- **Provider**: Управление состоянием.
- **SharedPreferences**: Локальное хранение данных.
- **Flutter Local Notifications**: Уведомления о дедлайнах и таймерах.
- **FL Chart**: Визуализация статистики.
- **Google Fonts**: Современная типографика.
- **Glassmorphism**: Современный UI дизайн.

## 🚀 Как собрать APK

1. Установите Flutter SDK.
2. Клонируйте репозиторий: `git clone https://github.com/winchestersdc79-hub/Student.git`
3. Выполните `flutter pub get`.
4. Соберите APK: `flutter build apk --release`.
