# Sviroxk Productivity App

## Обзор

Expo React Native приложение для продуктивности на русском языке. Полная замена Flutter-приложения с улучшенной функциональностью. Работает на Android, iOS и в браузере (Windows/macOS/Linux).

## Стек технологий

- **Монорепо**: pnpm workspaces
- **Node.js**: 24
- **Менеджер пакетов**: pnpm
- **TypeScript**: 5.9
- **Mobile**: Expo 54 + React Native 0.81
- **Навигация**: expo-router (файловая маршрутизация)
- **Хранилище**: AsyncStorage (локально на устройстве)
- **Состояние**: React Context + useReducer
- **API сервер**: Express 5 + PostgreSQL + Drizzle ORM
- **Уведомления**: expo-notifications 0.32.x
- **Медиа**: expo-image-picker (фото из галереи и камеры)

## Мобильное приложение (Sviroxk Productivity)

### Основные экраны (табы)
1. **Матрица** — Матрица Эйзенхауэра: 4 квадранта (Сделать / Запланировать / Делегировать / Убрать)
2. **Календарь** — Месячный вид с дедлайнами задач
3. **Фокус** — Pomodoro таймер с выбором задачи и трекингом времени
4. **Привычки** — Ежедневные привычки со стриками, рекордами и недельным графиком
5. **Статы** — Графики выполнения задач, фокус-сессий, статистика привычек

### Дополнительные экраны
- **Детали квадранта** — Список задач с сортировкой
- **Поиск** — Фильтрация по приоритету, квадранту, тексту
- **Архив** — Выполненные задачи с возможностью восстановления
- **Настройки** — Pomodoro, уведомления, сброс данных

### Функции задач
- Создание с квадрантом, приоритетом (П1/П2/П3), дедлайном, тегами, подзадачами
- **Прикрепление фото** (до 4 фото) из галереи или камеры
- Редактирование через долгое нажатие → контекстное меню
- Закрепление, перемещение между квадрантами, удаление
- Уведомления о дедлайнах (за день и в день)

### Уведомления (Android/iOS)
- **Дедлайны**: за 1 день до и в день дедлайна
- **Привычки**: ежедневное напоминание в настраиваемое время
- **Pomodoro**: уведомление по окончании сессии
- Запрашивает разрешение при первом запуске с объяснением
- Android: отдельные каналы (дедлайны / привычки / pomodoro)

### Поддержка платформ
- **Android**: APK через EAS Build (GitHub Actions)
- **iOS**: через EAS Build
- **Web/Windows**: браузерная версия с адаптивным дизайном (макс. ширина 768px)

## GitHub Actions (APK билд)

Workflow: `.github/workflows/build-apk.yml`

- Запускается при каждом push в `main` или вручную
- Устанавливает Android SDK + Java 17
- Собирает APK локально через `eas build --local`
- Создаёт GitHub Release с APK файлом
- Требует: `EXPO_TOKEN` в GitHub Secrets

## Архитектура

### Ключевые файлы
- `artifacts/mobile/contexts/AppContext.tsx` — Центральное состояние (задачи, привычки, сессии)
- `artifacts/mobile/app/(tabs)/` — Табы (index, calendar, timer, habits, stats)
- `artifacts/mobile/app/quadrant/[id].tsx` — Детали квадранта
- `artifacts/mobile/components/` — Компоненты (TaskCard, HabitCard, AddTaskModal, EditTaskModal)
- `artifacts/mobile/hooks/useNotifications.ts` — Уведомления
- `artifacts/mobile/app/_layout.tsx` — Корневой layout + setup уведомлений

## Команды

```bash
# Запуск мобильного приложения
pnpm --filter @workspace/mobile run dev

# Typecheck
pnpm run typecheck

# Сборка APK (требует EAS CLI и Expo аккаунт)
cd artifacts/mobile && eas build --platform android --profile preview --local
```

## Пользовательские настройки

- Весь UI на русском языке
- Тёмная тема: фон #0D0D1F, карточки #161630, акцент #A855F7 (фиолетовый)
- Данные хранятся локально (AsyncStorage) — не требует сервера для основных функций
